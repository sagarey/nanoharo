**5. 修改 sendMessage()（内存队列，不写文件）：**
```typescript
sendMessage(groupJid: string, text: string): boolean {
  const state = this.getGroup(groupJid);
  if (!state.active || state.isTaskContainer) return false;
  state.idleWaiting = false;
  state.pendingFollowUps.push(text);
  return true;
}
```
注意：删除原来检查 `!state.groupFolder` 的条件（内存队列不需要 groupFolder）。

**6. 修改 closeStdin()：**
```typescript
closeStdin(groupJid: string): void {
  const state = this.getGroup(groupJid);
  if (!state.active || !state.session) return;
  state.session.close();
}
```
删除原来写 `_close` sentinel 文件的逻辑。

**7. 修改 notifyIdle()（行为不变，但触发 session.close() 替代写文件）：**
逻辑已通过 closeStdin 改造正确处理，无需单独修改 notifyIdle。

**8. 修改 runForGroup() finally 块：**

**10. 修改 shutdown()：**
```typescript
async shutdown(_gracePeriodMs: number): Promise<void> {
  this.shuttingDown = true;

  const activeSessions: string[] = [];
  for (const [jid, state] of this.groups) {
    if (state.session) {
      activeSessions.push(jid);
      state.session.close();
    }
  }

  );
}
```
